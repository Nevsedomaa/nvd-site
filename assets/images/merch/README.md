# Как называть фото мерча

Рекомендуемый формат: `id-товара-номер.jpg`.

Примеры для товара `black-tshirt-nvd`:

- `black-tshirt-nvd-1.jpg`
- `black-tshirt-nvd-2.jpg`
- `black-tshirt-nvd-3.jpg`

Потом пропиши эти пути в `js/merch-data.js`:

```js
images: [
    "assets/images/merch/black-tshirt-nvd-1.jpg",
    "assets/images/merch/black-tshirt-nvd-2.jpg",
    "assets/images/merch/black-tshirt-nvd-3.jpg"
]
```
